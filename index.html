Anweisungen für die Bildintegration:

1. Bilder ausschneiden: Das Referenzbild in einzelne Komponenten aufteilen

2. Bilder optimieren: Für Web optimieren (WebP/JPG, komprimiert)

3. Platzhalter ersetzen: Die Divs mit image-placeholder Klasse durch <img> Tags ersetzen

4. Responsive Attribute: class="img-fluid" für automatische Größenanpassung hinzufügen
